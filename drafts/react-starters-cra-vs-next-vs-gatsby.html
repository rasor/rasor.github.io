React starters: CRA vs Next vs Gatsby

Posted on January 01, 2099 in Develop

Next.js vs Gatsby vs create-react-app

  • Create a New React App – Recommended Toolchains
  • The Next.js Handbook
    • CRA: CLI When you don't need Server-Side-Rendering (SSR)
    • Gatsby: When you don't deploy a backend (static site), but API is somewhere else
    • CNA - Next: Cli When you do need Node.js backend
    • Neutrino
      • Both for Apps and Libs
      • React, Preact, Vue, Web
    • Parcel - for web
    • Razzle
    • CRL: CLI For creating React component libraries with (or without) TypeScript
  • When to Use Static Generation (Gatsby) v.s. Server-side Rendering (Next)

Individual starters

  • CRA - Create React App
    • App CLI: npx create-react-app my-app --template typescript --use-npm - Getting Started
    • App CLI: npx create-react-app my-app --template redux-typescript --use-npm - cra-template-redux-typescript
  • CRA using Ionic components
    • App CLI: bash npm install -g @ionic/cli ionic start my-app tabs --type=react --no-deps
  • GatsbyJS
    • App CLI:
      bash npm install -g gatsby-cli gatsby new my-app
    • Starter templates
  • CNA - Next.js
    • App CLI: npm init next-app nextjs-blog --example "https://github.com/zeit/next-learn-starter/tree/master/learn-starter --use-npm" - Create a Next.js App
    • Guide: The Next.js Handbook
  • CRL - create-react-library
    • Lib CLI: npx create-react-library --template typescript --mananger npm - create-react-library
  • Neutrino
    • Lib CLI: npx @neutrinojs/create-project my-lib- @neutrinojs/react-components - React Components - Neutrino
    • App CLI: npx @neutrinojs/create-project my-app - @neutrinojs/react - React - Neutrino
  • Parcel

Alternative CLI starting

# install CLI
npm install -g create-react-app
# and use CLI
create-react-app my-app

# or use npx
npx create-react-app my-app

# or use npm
npm init react-app my-app

# or use yarn
yarn create react-app my-app

The end
