Install

PreReqs

See list on github: * VS Code * FrontEnd * Node.js * Backend * VS Code plugins: * C# for Visual Studio Code * SQL Server (mssql) * .NET Core SDK: Download .NET Core 3.1 SDK (v3.1.402) - Windows x64 Installer * SQL Express * Connection string: Server=localhost\SQLEXPRESS;Database=master;Trusted_Connection=True; * Verify: ps1 # From CMD shell: sqlcmd -S localhost\SQLEXPRESS -E select @@Version go # Microsoft SQL Server 2019 (RTM) - 15.0.2000.5 (X64) * Optionally Configure Windows Firewall * SQL Server Management Studio (SSMS) - SQL Server Management Studio (SSMS)

Run code

  • Open VSCode in folder .\ASP.NET-Core-3-and-React\Chapter08
# Terminal for Backend:
cd Chapter08
cd backend
dotnet restore
dotnet run
