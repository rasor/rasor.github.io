Ionic upgrade check-list

Posted on January 01, 2099 in misc

  • From v3.4.2

  • To v3.7.1

  • Create newest template and test

ionic start newesttemplate blank --type=ionic-angular
cd newesttemplate
ionic serve

  1. Diff the template with your old version

  2. Git updates

  3. In .gitignore add lines

.sourcemaps/
www/

If you have committed files in www\ you need to remove tracked files git rm -r --cached

  1. Dependencies - package.json

  2. Replace dependencies and devDependencies

  3. Extra added (project specific) dependencies you need to move and upgrade version manually

  4. src/service-worker.js

  5. After main.js add

    './build/vendor.js',

... or just replace the whole file.

  1. src/index.html

  2. Before cordova.js add

  <!-- add to homescreen for ios -->
  <meta name="apple-mobile-web-app-capable" content="yes">
  <meta name="apple-mobile-web-app-status-bar-style" content="black">
  • Before main.js add
  <!-- The vendor js is generated during the build process
       It contains all of the dependencies in node_modules -->
  <script src="build/vendor.js"></script>
