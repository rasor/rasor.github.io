Github Dependabot

Posted on January 01, 2099 in Develop

In github when you have dependabot PRs:

  • Find Dependabot PRs
    • Enter the PRs on https://github.com///pulls
    • Enter a specific PR e.g. https://github.com///pulls/6
  • Visibly Check the change
    • Expand Commits
    • Compare view
  • Accept the PR
    • Merge PR
    • Confirm
  • Pull to local repo
    • In VS Core
      • Files: Open package-lock.json
      • install GitLens
      • GitLens
        • Reposities: Pull
        • File History: Refresh
        • Notice Depandabots changes is now in the history
  • Update the local packages
    • In VS Core ```bash

    Fetch the updated dependencies

    npm install

    Test

    npm start ```
