AWS - Getting started

Posted on January 01, 2099 in DevOp

AWS getting started

  • Enable Multi Factor Authentication (MFA) via https://console.aws.amazon.com/iam/home
  • Use CloudFormation to create resources. You do that in Stacks via templates (CFT). When you have created a stack of resources, then you can delete it again - thus easy turn if off (and save your money).
    • Tip: You can generate a CFT from existing resources via CloudFormer
  • Test1 : Create a user having keys and groups memberships via IAM CFT

Prices

If you live in Europe region Ireland is currently marginal cheeper than Frankfurt for EC2 Spots.
Spots are good for short time provisionings.
You should also choose Linux. You can run Docker containers containing Windows or you could run .NET Core, if you require Microsoft SW.

You can save much extra by running Serverless.
